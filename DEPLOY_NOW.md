# 🚀 Deploy to GitHub Pages NOW

## Quick Steps (2 minutes):

### Step 1: Rename the file
Go to: https://github.com/brokenspagheti/wanda-network-state-city/blob/main/index-updated.html

1. Click the **pencil icon** (Edit this file)
2. Change the filename from `index-updated.html` to `index.html`
3. Scroll down and click **Commit changes**

### Step 2: Enable GitHub Pages
Go to: https://github.com/brokenspagheti/wanda-network-state-city/settings/pages

1. Under "Source", select **Deploy from a branch**
2. Under "Branch", select **main** and **/ (root)**
3. Click **Save**

### Step 3: Wait 1-2 minutes
GitHub will build and deploy your site automatically.

### Step 4: Visit your live site!
Your Minecraft Network State City will be live at:
**https://brokenspagheti.github.io/wanda-network-state-city/**

## Troubleshooting:

**If the site doesn't load:**
- Wait 2-3 minutes for GitHub to build
- Check Settings > Pages shows "Your site is live at..."
- Clear browser cache (Ctrl+Shift+R)

**If you see 404:**
- Make sure the file is named exactly `index.html` (not `index-updated.html`)
- Make sure GitHub Pages is enabled in Settings > Pages
