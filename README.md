# 🌐 Wanda - Network State City

An interactive 3D visualization of **Wanda**, a futuristic network state city built with Three.js. This project showcases a decentralized urban planning concept with interconnected districts, digital infrastructure, and sustainable design.

![Wanda City](https://img.shields.io/badge/Three.js-0.160.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## 🏙️ City Districts

### 🏛️ Governance Hub
- Central governance tower (cyan glow)
- 6 surrounding council buildings
- DAO operations & digital democracy center

### 💼 Innovation District
- Tech campus with 6 buildings
- Startup incubation spaces
- Technology development zone

### 🏘️ Residential Nodes
- 6 decentralized living clusters
- 24 residential buildings total
- Distributed community design

### 🌳 Green Commons
- 4 sustainable parks
- Green spaces with trees
- Public recreation areas

### 📡 Data Centers
- 4 corner infrastructure nodes
- Network backbone with antennas
- Digital infrastructure hubs

## ✨ Features

- **Interactive 3D Environment**: Drag to rotate, scroll to zoom
- **Dynamic Lighting**: Pulsing buildings with cyberpunk aesthetics
- **Network Visualization**: Glowing connections between districts
- **Particle System**: 1000 ambient particles
- **Responsive Design**: Adapts to any screen size
- **Fog Effects**: Atmospheric depth

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/brokenspagheti/wanda-network-state-city.git
cd wanda-network-state-city
```

2. Open `index.html` in your browser:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve

# Or simply open index.html in your browser
```

3. Navigate to `http://localhost:8000`

## 🎨 Design Concept

Wanda represents a **Network State** - a digitally-native community with:

- **Decentralized Governance**: Central hub with distributed council
- **Innovation Focus**: Dedicated tech development zones
- **Distributed Living**: Multiple residential clusters
- **Sustainability**: Integrated green spaces
- **Digital Infrastructure**: Robust network backbone
